# SaaS Account

&#x20;The Account tab allows the user to review details relating to their account including _**User Name**_, _**User Role**_, _**User Email**_, and the ability to change their _**password.**_

![SymetryML SaaS: Account information](<../.gitbook/assets/saas\_change\_password\_1.png>)

When clicking on the _**Change**_ tab, the following screen appears detailing the new password that has been generated which the user can copy and retain for future use.&#x20;

![SymetryML SaaS: Account generating new password](<../.gitbook/assets/saas\_change\_password\_2.png>)
