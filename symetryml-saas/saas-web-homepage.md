# SaaS Homepage

When logging into the SymetryML SaaS environment the user arrives at the following homepage. &#x20;

![SymetryML SaaS: Homepage](<../.gitbook/assets/saas\_web\_home.png>)

From here you are presented with three options, as detailed below:

_**Go to Modelling & Analytics UI**_ -  Select this option and a small pop-up window appears displaying all running instances, allowing the user to select the instance they want to connect to.

![SymetryML SaaS: Go To Modelling & Analytics UI](<../.gitbook/assets/saas\_quick\_connect.png>)

_**Go To REST API**_ - This option takes the user to SymetryML REST API documentation.

![SymetryML SaaS: Connecting to SymetryML REST API documentation](<../.gitbook/assets/Screen Shot 2022-06-21 at 12.08.50 PM.png>)

_**SML Capabilities & Documentation**_ - This selection takes the user to SymetryML documentation.&#x20;

![SymetryML SaaS: Connecting to SymetryML documentation](<../.gitbook/assets/Screen Shot 2022-06-21 at 12.08.17 PM.png>)
