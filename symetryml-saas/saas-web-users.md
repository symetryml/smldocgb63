# SaaS Users

The _**Users**_ tab keeps track of all existing users on the platform plus, when clicking on the _**Actions**_ tab, it allows _**administrators**_ and _**root**_ users to modify, add, and delete users to the SymetryML SaaS platform.&#x20;

![SymetryML SaaS: User modification, Addition, and Deletion](<../.gitbook/assets/saas\_users\_add.png>)

When you select a user, the _**User Instance**_ section populates with the current instances assigned to the user selected.&#x20;

![SymetryML SaaS: Instances assigned to users](<../.gitbook/assets/saas\_users\_instances.png>)

In the _**User Instance**_ section you are able to select _**Add Instance**_, _**Modify**_ and _**Remove**_ instance from a selected user.&#x20;

When selecting a user and clicking on _**Add Instance**_, the following window appears where the user is able to select an instance from a list of available instances and assign the selected instance to the selected user.

![SymetryML SaaS: Add Instance to selected user](<../.gitbook/assets/saas\_users\_add\_instance1.png>)

Click _**Add**_ and the selected instance will now be added to the selected user.

![SymetryML SaaS: Instance added to selected user](<../.gitbook/assets/saas\_users\_instances1.png>)

Select a user and then click on an instance assigned to the user.

Next, click _**Modify**_ and the following screen will appear where the user can modify the number of _**Core**_ of an instance they wish to allocate to a particular user.

![SymetryML SaaS: Allocate instance resources](<../.gitbook/assets/saas_users_modify.png>)
