# SaaS Licence

The SymetryML SaaS _**Licence**_ page displays the details of an organizations current SymetryML SaaS licence.&#x20;

![SymetryML SaaS: Licence Information](<../.gitbook/assets/saas\_license.png>)

This _**Licence**_ tab will display important information about the current licence including the _**Expiry Date**_, _**Available CPU/GPU**_, and the _**number of users**_ allowed.&#x20;
